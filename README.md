# The Egg Cafe

The Egg Cafe is a responsive website designed for a cafe shop.

## Architecture

- Application: HTML, CSS, Bootstrap

## Functions
#### Home
- Slideshow displaying highlights of the cafe
- Container grids showing best features of cafe
